# E-Commerce-API Video Game Store
### About
A fullstack web application debugged using Insomnia. It's designed to emulate an online shopping experience to buy video games online. Utilizes a REST Api built in Java and Spring Boot on the back end.
## How to use
- Have Java JDK 17 installed along with MySQL Server.
- Run the backend program in your IDE of choice.
- Go to http://localhost:8080 on your web browser to see the Spring Boot server
- Create a new profile via http://localhost:8080/swagger-ui.html
